# 🍽️ Restaurant Menu React App

A simple and responsive **Restaurant Menu application built with React**.
This project displays a list of food items and allows users to **filter the menu by category** such as Breakfast, Lunch, Dinner, Snacks, and Desserts.

The goal of this project is to practice **React fundamentals** including components, props, state management, and dynamic rendering.

---

## 🚀 Features

* Dynamic menu rendering using **React components**
* **Category filtering** functionality
* Built with **React Hooks (useState)**
* Clean and responsive UI
* Reusable components (Navbar, Card, Data)

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3

---

This project was created while learning:
